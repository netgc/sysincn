2022 年 5 月 11 日，IBM 收购的红帽宣布推出 Red Hat Enterprise Linux 8.6 (RHEL)，这是世界领先的企业 Linux 平台的新版本——组织可以在此基础上扩展现有应用程序并在裸机、虚拟、容器中推出新兴技术和所有类型的云环境。

RHEL 8.6 的新增功能

RHEL 8.6 包括广泛的新功能和增强功能，其中许多是直接响应我们的客户和合作伙伴的建议而做出的。

主要特点和优势

降低混合云中的潜在风险 通过帮助组织构建具有更高稳定性和信任度的平台，这为更强大的安全态势奠定了基础，以扩展应用程序并在组织需要的地方推出新兴技术。这些能力包括：

  • Web 控制台支持使用 Sudo 和 SSH 进行智能卡身份验证，以便在执行管理功能和通过 RHEL Web 控制台访问远程主机时使用智能卡身份验证凭据。这有助于通过更透明地利用存储在智能卡上的数字证书来简化管理和合规性。
  • 使用 Red Hat 和 SAP 支持生产中的 SAP HANA，这是一种启用 SELinux 的联合测试的 RHEL 配置。SELinux 的安全技术允许进程隔离，为 SAP 用户提供出色的权限提升攻击缓解。
  • 专为 SAP 解决方案设计的增强安全功能，帮助组织遵守更高的安全标准，同时仍提供最佳数据库性能。借助围绕 SAP 安全性的一系列新的最佳实践和功能，用户现在可以在 SAP HANA 部署的环境中进一步强化他们的操作系统。通过使用 SELinux 或网络绑定磁盘加密 (NBDE)，组织可以减少攻击面并更好地保护静态业务关键数据。

自动化和管理 使组织能够更轻松地最大限度地减少手动任务、标准化大规模部署并简化其系统的管理。主要改进包括：

  • Web 控制台（技术预览）中的 Stratis Storage 支持使用户能够使用 Red Hat Enterprise Linux Web 控制台创建、配置和管理 Stratis 存储池和文件系统。
  • 使用 Web 控制台将主机设备直通到虚拟机，使用户能够使用 Web 控制台将主机 PCI（外围组件互连）和 USB 设备直通到虚拟机。此功能允许主机系统上的这些设备在虚拟机中可用。
  • 高可用性 (HA) 集群系统角色，用于创建更一致和稳定的红帽企业 Linux HA 集群解决方案的自动化工具。组织现在可以利用规定的用例场景、配置和拓扑建议更轻松地创建 HA 集群。
  • Web 控制台的系统角色，用于自动安装和配置 Red Hat Enterprise Linux Web 控制台，包括指定应安装哪些 Web 控制台软件包、启动和启用 Web 控制台，以及可选地配置空闲超时等设置。
  • 防火墙系统角色使组织能够自动管理其防火墙配置。
  • SAP day-1 Automation 的系统角色为用户提供红帽 Ansible 自动化平台用于 SAP HANA 环境的自动化设置和配置过程。此外，这些新的红帽企业 Linux 系统角色现在可作为 Ansible 集合使用，为组织提供更多使用 SAP 自动化内容的灵活性。

通过增强边缘的工作负载弹性和安全性来解决边缘问题，主要更新包括：

  • 边缘管理可帮助组织以更高的边缘安全性来管理和扩展部署。零接触配置、系统运行状况可见性和快速安全修复的优势可从单个界面获得。这些功能在每个边缘系统的生命阶段提供控制和安心。
  • 简化的边缘安装程序，以更好地减轻在现场或集中位置部署远程系统时的管理负担，更加一致并具有更高的安全性。
  • 边缘载入支持 FIDO 设备载入 (FDO) 作为设备的安全标准，可自动执行配置后步骤并远程载入管理平台。
  • 通过红帽企业 Linux 的智能回滚框架进行默认健康检查，以测试每次操作系统更新的网络功能，并在发生故障时回滚 。

为开发提供更一致的基础 ，使开发人员能够自由和灵活地在混合云中更快地进行创新。主要语言和工具更新包括：

  • Perl 5.32 到 RHEL8 AppStream
  • PHP 8.0 到 RHEL8 AppStream
  • RHEL8 AppStream 中的 LLVM 工具集
  • RHEL8 AppStream 中的 Rust 工具集
  • RHEL8 AppStream 中的 Go 工具集
