使用 Elastic 8.5 版构建强大的搜索体验

使用 Elastic 8.5 版构建强大的搜索体验，缩短实现可观测性的时间，并简化安全工作流

作者 Dan Courcy，2022年11月1日

今天，Elastic 很高兴宣布，全球领先的搜索驱动型解决方案平台的最新版 Elastic 8.5 正式全面上线。

Elastic 8.5 版为 Elastic Search Platform 及其三个内置解决方案（Elastic 企业搜索、Elastic 可观测性和 Elastic 安全）引入了一系列新功能 (sysin)。

新增功能

Elastic 企业搜索是通过 Elasticsearch 实现强大的现代搜索和发现体验的黄金标准，可让您随时随地搜索任何内容。

  • 在 Elastic 8.5 版中，Elastic 企业搜索正式引入了高级矢量搜索功能，其中包括混合排名，此排名功能将矢量相似性与查询评分结合在一起。此外，原生 MongoDB 和 MySQL 集成有助于简化数据采集，而新的 Machine Learning 采集管道有助于简化数据扩充 (sysin)。
    进一步了解 Elastic 8.5 版在哪些方面改进了 Elastic 企业搜索。

Elastic 可观测性是部署最为广泛的解决方案，用于将指标、日志和痕迹转换为行之有效的 IT 见解，让您能够实现对整个数字生态系统的一体化可观测性。

  • 在 Elastic 8.5 版中，SRE 和 DevOps 工程师能够使用 Elastic 可观测性为他们环境中的现有 .NET 应用程序自动插桩。无论应用程序是在容器中运行、在 IIS 中运行，还是作为 Windows 或 Linux 服务运行，您只需短短几分钟就可以完全了解自己的整个 .NET 应用程序生态系统，而无需重新编译任何二进制文件。
    进一步了解 Elastic 8.5 版在哪些方面改进了 Elastic 可观测性。

Elastic 安全集 SIEM、Endpoint Security 和云安全于一体，可帮助从业人员在云范围内快速防御、检测和应对威胁。

  • 在 Elastic 8.5 版中，Elastic 安全引入了新的威胁情报管理维度，有助于加快调查、响应和恢复速度。通过让 SOC 能够利用这一背景信息，有助于增强该解决方案中快速发展的 SOAR 功能，为现代化的 SIEM 注入强劲动力。
    进一步了解 Elastic 8.5 版在哪些方面改进了 Elastic 安全。

Elastic 的搜索驱动型解决方案均基于一个统一的堆栈（即 Elastic Stack）构建而成，通过 Elastic Cloud 在 AWS、Microsoft Azure 和 Google Cloud 上以托管服务的形式提供。升级到 Elastic 8.5 版后，您可以使用 HNSW（现已正式发布）的矢量搜索功能更智能地进行搜索，更快地诊断日志速率峰值，并分配案例以便于协作。Elastic Cloud 8.5 版还提供了一个新的运行状况页面，可以轻松识别和解决集群性能问题。进一步了解 Elastic 8.5 版在哪些方面改进了 Elastic Stack 和 Elastic Cloud。

即刻体验

准备好升级了吗？将数据转化为行之有效的见解从未如此简单。Elastic 8.5 版现已在 Elastic Cloud 上正式推出，这是唯一一个包含此最新版中所有新功能的托管型 Elasticsearch Service。
