Parallels Desktop 16.5 for Mac 已支持 M1 和 Intel 两种芯片

2021 年 4 月 14 日 Parallels® Desktop 16.5 for Mac 正式发布了，它对使用 Apple M1 或 Intel 芯片的 Mac 计算机提供全面的本机支持。用户现在将可以在 ARM Insider Preview 以及 Apple M1 Mac 计算机上最受欢迎的基于 ARM 的 Linux 发行版上运行 Windows 10 及其应用程序。

Parallels Desktop 16.5 为在 M1 芯片的 Mac 计算机上提供无缝的「Mac 上的 Windows」体验树立了新标准，这样完美的搭配可在个人计算机上提供世界上最快的集成显卡 (sysin)，完成了革命性的能效，并且也旨在完全支持 macOS Big Sur.

Parallels 工程与支持高级副总裁 Nick Dobrovolskiy 表示：“对于 Mac 用户而言，Apple 的 M1 芯片是一个重大突破。借助 Rosetta 技术，大多数 Mac 应用程序的过渡非常顺利。因此，Parallels 工程师为带有 M1 芯片的 Mac 实现了本机虚拟化支持。这使我们的用户能够享受到最佳的「Mac 上的 Windows」体验。”

超过十万名 M1 Mac 用户参与测试了技术预览版

我们非常感激这十万多个测试了 Parallels Desktop 16.5 for M1 Mac 技术预览版的 M1 Mac 用户以及他们的反馈。他们在 ARM Insider Preview 上运行 Microsoft 的 Windows 10 系统，以及成千上万种基于 Intel 的 Windows 应用程序，包括 Microsoft Office for Windows，Microsoft Visual Studio，SQL Server，Microsoft PowerBI 和 MetaTrader。

Dobrovolskiy 说：“我们从用户那里收到了对于以下测试对象出色性能的热烈反馈：Parallels Desktop 16.5 for M1 Mac 技术预览，在 ARM Insider Preview 上的 Windows 10 以及 x86 应用程序和游戏（包括 Rocket League，Ab Us，Roblox，The Elder Scrolls V：Skyrim，Sam 和 Max 拯救世界等等。） 测试人员们喜欢 Parallels Desktop 的易用功能以及 Windows 与 macOS Big Sur 的无缝集成 (sysin)。这大大提高了我们的开发效率。”

应用程序运行更快，更高效

使用 Parallels Desktop 16.5 for Mac，用户可以从 Apple 的 M1 芯片性能改进中受益，并体验更快，更高效地运行的应用程序。性能方面的提升包括：

• 能耗降低多达 250％： 在配备 Apple M1 芯片的 Mac 上，Parallels Desktop 16.5 的能耗比 2020 年基于 Intel 的 MacBook Air 计算机少 2.5 倍。

• DirectX 11 性能提高多达 60％： 在 M1 Mac 上运行的 Parallels Desktop 16.5 与具有 Radeon Pro 555X GPU 的基于 Intel 的 MacBook Pro 相比，DirectX 11 性能提高了 60％。

• 虚拟机性能（Windows）最高提高 30％： 在 M1 Mac 上的 Parallels Desktop 16.5 上的 ARM Insider Preview 中运行 Windows 10 的虚拟机（VM）的性能比在基于 MacBook Pro 的英特尔酷睿 i9 处理器上运行的 Windows 10 VM 的性能高 30％。

“我在 M1 Mac Mini 的 ARM Insider Preview 上安装了 Parallels Desktop 16.5 for Mac 和 Windows 10，因为作为数学老师，我平时使用的多个软件应用不能继续在 M1 Mac 上运行，” 来自美国新墨西哥州阿尔伯克基的 Daniel Dudley 说道，“在 Parallels Desktop 16.5 中的 M1 Mac 上运行 Windows 真是棒极了 (sysin)。这是我所拥有过的最快的 Windows 版本！”

在 M1 Mac 计算机上运行 Linux 发行版

使用 Parallels Desktop 16.5 for Mac，用户不仅可以在 M1 Mac 计算机上的虚拟机中的 ARM Insider Preview 上运行 Windows 10，而且还可以使用 Linux 发行版 Ubuntu 20.04，Kali Linux 2021.1，Debian 10.7 和 Fedora Workstation 33-1.2。

“我爱上了能在我的 M1 Mac 上的 Parallels Desktop 16.5 中运行 Linux 环境的感觉，” 英国 Darren Paxton 说。“性能简直令人赞叹。”

适用于 M1 Mac 的 Parallels Desktop 16.5 的最佳功能

了解为 Mac M1 芯片重新设计的一些 Mac Parallels Desktop 16 最佳功能：

• 融合模式：在 Mac 上运行 Windows 应用程序就像在 Mac 上运行本机系统一样，无需管理两个单独的桌面或重新启动。

• 共享配置文件：与虚拟机共享 Mac 桌面，图片，文档和其他文件夹，以便于访问。

• 触摸栏控件：快速访问虚拟机中的应用程序，并自定义触摸栏快捷方式以使用最常用的应用程序和命令。

• Mac 键盘布局：轻松自定义键盘菜单和快捷方式，以获得更个性化的体验。

• 包含数百种其他 Parallels Desktop 功能，更有助于提高客户的工作效率。

• 在虚拟机中使用 macOS Big Sur 是 Parallels 希望在今年晚些时候添加的功能。

对于 Parallels Desktop 16.5 针对基于 Intel 的 Mac 计算机的广泛功能的所有要点均可以在我们官网上查看。基于 Intel 的 Parallels Desktop for Mac 用户可以运行超过 50 种受支持的来宾系统，包括 Windows，macOS，OS X，Linux 和 Android OS 的多个版本，以及数十种其它版本。

如何获取 Parallels Desktop 16.5 的副本

如果你…

… 是现有客户：拥有任意版本的 Parallels Desktop 16 for Mac 许可证的人都可以免费自动更新到 Parallels Desktop 16.5。

… 是一个新客户：Mac 的所有 Parallels Desktop 16.5 更新版本都支持 M1 和 Intel Mac 计算机（标准版，专业版和商业版）。在 www.parallels.cn 购买您的副本 - 或免费试用 14 天。

注意：在 Parallels Desktop 16.5 中的 Apple M1 Mac 计算机上运行 VM 需要基于 ARM 的操作系统（OS）。在 Parallels Desktop 虚拟机上安装来宾操作系统的客户有责任确保它们符合每个操作系统的最终用户许可协议（EULA）。
