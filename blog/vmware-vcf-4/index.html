什么是 VMware Cloud Foundation？

VMware Cloud Foundation 是基于全栈超融合基础架构 (HCI) 技术构建的混合云平台，用于管理虚拟机和编排容器。凭借易于部署的单一体系架构，VMware Cloud Foundation 可在私有云和公有云之间实现一致、安全的基础架构和运维 (sysin)。通过可以提供这一切的混合云，提高企业的敏捷性和灵活性。

VMware Cloud Foundation 的优势

简化的管理

通过集成的容器编排（由 vSphere 7 with Kubernetes 提供支持）优化 Kubernetes 集群的性能、恢复能力和可用性。

灵活的部署选项

跨私有云和公有云访问极为广泛的部署选项，并从自管理 HCI 和 HCI 即服务选项中进行选择。

增强的应用性能

通过高性能存储和跨每个环境的无缝资源池的负载均衡 (sysin)，提高应用的性能和可靠性。

标准化体系架构

使用您在数据中心使用的一流的计算、存储、网络连接和管理工具自动执行 HCI 交付，以简化运维并降低 TCO。

最大化规模和 IT 效率

通过在各处扩展相同的基础架构、运维、工具和流程，管理本地部署或云中的工作负载。

原生安全性

通过我们的全栈 HCI 平台确保企业级安全性，该平台整合传统虚拟机和现代容器工作负载。

VMware Cloud Foundation 的应用场景

私有云和混合云

从本地部署、边缘或云环境中的虚拟化资源池调配基础架构。自动化和编排简化了基于虚拟机或容器的工作负载的管理，这些工作负载可以扩展以满足您的业务需求。

现代应用

通过 VMware Cloud Foundation Services（一种集成了 Kubernetes 和 RESTful API 的新界面）在本地或云中开发和部署现代应用。

虚拟桌面基础架构

实现 Horizon 7 虚拟桌面基础架构 (VDI) 的自动化部署，以标准化、自动化和简化虚拟桌面。

常见问题解答

查找有关 VMware Cloud Foundation 的一些常见问题的答案

  • 哪些版本的 VMware Cloud Foundation 包含集成的 vSphere with Kubernetes？

Standard、Advanced 和 Enterprise 版 VMware Cloud Foundation 包含 vSphere 和 Tanzu Kubernetes Grid 功能。VMware Cloud Foundation Starter 版包含标准 vSphere 虚拟机功能。

  • 何时支持 VMware Cloud Foundation 早期版本的升级？

VMware Cloud Foundation 4.0 仅限于全新 VCF 环境。目前不支持对现有 VCF 部署进行升级。现有的 VMware Cloud Foundation 实例可以从 3.9 升级到 3.9.1，在未来版本中还将支持升级到 4.x。

  • 如何将 VMware Cloud Foundation 用于混合云？

VMware Cloud Provider 计划 (VCPP) 中的精选服务提供商提供由 VMware Cloud Foundation 支持的云服务，这些提供商包括 CenturyLink、OVH 和 Rackspace。VMware Cloud on AWS 是 VMware 运维、管理和销售的按需服务，由 VMware Cloud Foundation 提供支持。其他解决方案包括 Azure VMware Solutions by CloudSimple 和 Google Cloud VMware 解决方案。

  • 如何购买 VMware Cloud Foundation 软件？

购买 VMware Cloud Foundation 有四种方式：(1) 直接从 VMware 购买；(2) 通过 VMware 渠道合作伙伴购买；(3) 作为集成系统的一部分通过 OEM 供应商购买；以及 (4) 作为订阅服务通过公有云服务提供商购买。

如要了解更多问题及其相应答案，请查看完整版的 VMware Cloud Foundation 常见问题解答。
