SQL 2017 功能

行业领先的性能。

在性能（尤其是大规模性能）价格比方面位居第一

SQL Server 在 TPC-E OLTP 工作负载、TPC-H 数据仓库工作负载和实际应用程序性能基准方面始终处于领先地位。现已在 Windows 和 Linux 上实现破纪录的性能。

性能最高的数据仓库

获得对小型数据市场到大型企业数据仓库的支持，同时通过增强数据压缩来降低存储需求。将企业级关系型数据仓库扩展到可容纳 PB 级数据——并与 Hadoop 等非关系型数据源集成。

​

漏洞最少的数据库。

安全性和合规性

利用漏洞最少的数据库保护静态和动态数据（据美国国家标准与技术研究院的美国国家漏洞数据库 2017 年 1 月 17 日发布的数据，我们的数据库在 NIST 漏洞数据库连续 6 年被评为任何主流平台上漏洞最少的数据库）。SQL Server 2017 中的安全创新通过一种多层保护的安全方法帮助保护任务关键型工作负载的数据，这种方法在行级别安全性、动态数据掩码、透明数据加密 (TDE) 和可靠审核的基础上添加了始终加密技术。

​

任务关键型可用性。

高可用性和灾难恢复

SQL Server 2017 中增强的 AlwaysOn 是一个用于在 Linux 和 Windows 上实现高可用性和灾难恢复的统一解决方案，利用它可获得任务关键型正常运行时间、快速故障转移、轻松设置和可读辅助数据库的负载平衡。此外，在 Azure 虚拟机中放置异步副本可实现混合的高可用性。

​

端对端移动商业智能。

企业商业智能

利用全面的商业智能 (BI) 解决方案扩展 BI 模型，丰富数据，并确保质量和准确性。SQL Server Analysis Services 帮助您构建全面的企业级分析解决方案，并从内置于表格模型的 in-memory 快如闪电的性能中受益。针对表格模型和多维模型使用直接查询以缩短获得见解的时间。

任何设备上的端到端移动商业智能

借助新式分页报告和丰富的可视化获得见解并实现业务转型。使用 SQL Server Reporting Services 向任何移动设备（包括 Windows、Android 和 iOS 设备）发布报表并在线或离线访问报表。

简化大型和小型数据

利用 PolyBase 技术（此技术使用简单的 T-SQL 命令查询 Hadoop）将关系型数据与大数据结合起来。JSON 支持可让您分析和存储 JSON 文档并将关系型数据输出到 JSON 文件中。在 SQL Server 2017 中，现在可以管理和查询关系数据库内的图形数据。

​

内置分析。

数据库内高级分析

使用 R 和 Python 通过 SQL Server 机器学习服务构建智能应用程序。通过直接在数据库中执行高级分析，超越被动响应式分析，从而实现预测性和指导性分析。通过使用多线程和大规模并行处理，与单独使用开源 R 和 Python 相比，您将更快地获得见解。

实时混合事务性/分析处理

在 SQL Server 2017 中将 in-memory 列存储和行存储功能结合起来，从而实现实时运营分析，即，直接对事务性数据进行快速分析处理。开放实时欺诈检测等新方案，同时不会影响您的事务性能。

​

一致的体验。

现在可以在 Windows、Linux 和 Docker 上使用

开发一次，即可部署到本地及云等任何环境，提供一致的体验。现在支持 Windows 和 Linux 以及 Docker 容器。

从本地到云的一致数据平台

获得从本地到云的一致体验 - 让您构建和部署用于管理您数据投资的混合解决方案。从在 Azure 虚拟机中运行 SQL Server 工作负载的灵活性中获益，或使用 Azure SQL 数据库扩展并进一步简化数据库管理。

易于使用的工具和连接器

使用您已有的技能和熟悉的工具（例如 Azure Active Directory 和 SQL Server Management Studio）来管理您在本地 SQL Server 及 Microsoft Azure 中的数据库基础结构。跨各种平台应用行业标准 API 并从 Visual Studio 下载更新的开发人员工具，以构建下一代的 Web、企业、商业智能以及移动应用程序。
