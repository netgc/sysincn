什么是 Creative Cloud 创意应用软件？

Adobe Creative Cloud 创意应用软件将新版本的 Adobe 创意应用软件（包括 Photoshop、Illustrator 和 InDesign）带到了您的指尖。您可以自行决定其部署方式和时间。

Adobe Creative Suite 6 发布以来，我们已增加 1000多种新功能，旨在提高生产力、支持新的标准和硬件，并简化日常任务 (sysin)。如果您仍在使用 Creative Suite，请立即转换使用创意应用软件 Creative Cloud，切勿错失良机。

专业高效的创意应用软件

Creative Cloud 创意应用软件可以满足您的各种业务需求，包括修改照片、设计图形和插图、编辑视频等等。

  • 其他 Adobe 应用程序何时可以在 Apple M1 计算机上本地运行？

    我们正在努力确保其他应用程序能够在此类计算机上本地运行，但目前尚未确定所有这些应用程序的发布日期。如果有更多详细信息，我们将更新此文档。

  • 使用 Rosetta 2 运行的应用程序

    您可以在采用 Rosetta 2 技术的 Apple M1 计算机上安装以下应用程序。Rosetta 2 会对这些应用程序进行转换以在 M1 计算机中使用，因此初次打开这些应用程序可能需要较长时间。

    Adobe Animate

    Adobe Bridge（已知有问题）

    Adobe Premiere Elements 2022

    Adobe Photoshop Elements 2022

    Substance 3D Designer

    Substance 3D Painter

    Substance 3D Sampler

    Substance 3D Stager

    ​

    **注意：**在部署企业版应用程序或在使用独立安装程序时，确保已在用户计算机上装有 Rosetta 2。

  • 能否在 M1 计算机上安装 Rosetta 2？

    在 M1 计算机上启动任何已安装的基于 Intel 的应用程序即可安装 Rosetta 2。此外，还可打开终端并运行以下命令：

    • 如果有 root 权限，则运行：/usr/sbin/softwareupdate --install-rosetta

    • 如果没有 root 权限，则运行：/usr/sbin/softwareupdate --install-rosetta --agree-to-license

  • 可在何处下载应用程序？

    您可以直接从 creativecloud.adobe.com/apps 将这些应用程序下载到 M1 设备上。Creative Cloud 桌面应用程序会自动随应用程序一起安装。

  • 能否在 Apple M1 计算机上安装适用于 Creative Cloud 应用程序的插件？

    是的，可从 Creative Cloud 桌面应用程序中的“图库和市场”选项卡在 Apple M1 计算机上安装插件或扩展。

    **注意：**Photoshop 仅在 Apple M1 计算机上原生地支持新插件（基于 UXP，在菜单中的插件下）。如果要使用旧版插件（在菜单中的窗口下），请在 Rosetta 模式下运行 Photoshop。如果您是开发人员，请检查是否可以将插件从 CEP 迁移到 UXP。

  • Adobe 是否支持使用 Rosetta 2 运行的应用程序？

    我们为上面列出的应用程序提供支持。我们尚未对其他 Adobe 应用程序进行认证，不为在 M1 设备上使用那些应用程序提供支持。为获得最佳效果，请仅在系统要求中规定的操作系统和硬件平台上运行所有其他 Adobe 应用程序。

系统要求

Adobe Creative Cloud 2022 系统要求：

  • macOS Intel x64
    macOS Catalina (version 10.15) or later | recommended macOS Big Sur (version 11) macOS Catalina (version 10.15)

  • macOS Apple Silicon
    macOS Big Sur (version 11.2.2) or later

  • Windows
    Windows 10 (64-bit) version 1909 or later; LTSC versions are not supported

Adobe Creative Cloud 2021 系统要求：

  • macOS Intel x64
    macOS Mojave (version 10.14) or later | recommended macOS Big Sur (version 11) macOS Catalina (version 10.15)

  • macOS Apple Silicon
    macOS Big Sur (version 11.2.2) or later

  • Windows
    Windows 10 (64-bit) version 1809 or later; LTSC versions are not supported

Adobe Creative Cloud 2020 系统要求：

  • macOS
    macOS version 10.13 (High Sierra), macOS version 10.14 (Mojave), macOS version 10.15 (Catalina) (macOS 10.13.6 or later is recommended for best performance)

  • Windows
    Microsoft Windows 7 with Service Pack 1 (64-bit), Microsoft Windows 10 October 2018 update (64-bit) version 1809 or later
